# Pass Safe

## 프로젝트 개요

### 1-1. 한 줄 소개

사용자가 자신의 비밀번호가 안전한지 검사하고, 강력하고 안전한 비밀번호를 생성 및 관리할 수 있는 서비스

### 1-2. 배경 및 내용

비밀번호는 해커들에게 가장 빈번히 공격받는 대상 중 하나로, 마이크로소프트의 '2022년 디지털 방어 보고서'에 따르면 비밀번호 해킹 공격은 초당 921번 발생하고 있습니다. 공격자들은 무작위 대입, 유출된 계정 정보 활용, 피싱 등의 방법으로 비밀번호를 노리며, 이러한 다양한 공격으로부터 계정을 안전하게 보호하려면 자신이 사용하는 비밀번호의 안전성을 점검하는 것이 필수적입니다.

**Pass Safe**는 이러한 문제를 해결하기 위해 사용자가 비밀번호 보안을 쉽게 점검하고 관리할 수 있는 솔루션을 제공합니다. 'zxcvbn' 라이브러리를 활용하여 비밀번호의 복잡성과 취약성을 분석하며, 'Have I Been Pwned' API를 통해 데이터 유출 여부를 확인하는 기능을 통합합니다. 이를 기반으로 안전한 비밀번호를 생성하고 관리할 수 있는 환경을 제공하며, 디지털 보안을 한층 강화하는 것을 목표로 합니다.

### 1-3. 목표

Pass Safe는 오픈소스 프로젝트로, 누구나 신뢰할 수 있는 비밀번호 보안 점검 및 관리 솔루션을 제공하는 것을 목표로 합니다. 비밀번호 안전도 검사와 생성 기능을 모듈화하여 구현하며, 전 세계 개발자와 보안 전문가들이 검증하고 확장 가능한 환경을 제공합니다.

## 요구 사항 (임시)

### 2-1. 제공 기능

1. **비밀번호 안전도 평가**:  
   사용자가 입력한 비밀번호의 강도와 예상 해킹 시간을 분석하여 시각적으로 결과를 제공.
2. **비밀번호 유출 확인**:  
   'Have I Been Pwned' API를 통해 비밀번호 유출 여부를 확인.
3. **안전한 비밀번호 생성**:  
   강력하고 안전한 비밀번호를 사용자의 요구에 맞게 생성.
4. **생성한 비밀번호 관리**:  
   안전한 암호화 방식을 통해 비밀번호 저장 및 관리.
